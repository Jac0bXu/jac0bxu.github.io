# jac0bxu.github.io

Personal website repository for GitHub Pages.

## Structure

This repository follows the structure of a personal website with the following directories:

- `categories/` - Blog post categories
- `css/` - Stylesheets
- `me/` - About page
- `posts/` - Blog posts
- `tags/` - Post tags

## Setting Up GitHub Pages

1. **Push your code to GitHub:**
   ```bash
   git add .
   git commit -m "Initial commit"
   git push origin main
   ```

2. **Enable GitHub Pages:**
   - Go to your repository on GitHub
   - Click on **Settings**
   - Scroll down to **Pages** in the left sidebar
   - Under **Source**, select **Deploy from a branch**
   - Choose **main** branch and **/ (root)** folder
   - Click **Save**

3. **Your site will be live at:**
   - `https://jac0bxu.github.io/`

   Note: It may take a few minutes for the site to be published after enabling GitHub Pages.

## Customization

- Edit `index.html` to customize your homepage
- Add your content to `me/index.html` for the about page
- Add blog posts to the `posts/` directory
- Customize styles in `css/style.css`
- Update `sitemap.xml` and `index.xml` as you add content
